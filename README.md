# Audio Latency & Speech Synthesis Research Lab (v1.43)

## Project Overview
Este repositório é dedicado ao estudo experimental de latência em sistemas de feedback sonoro baseados em navegador. O foco principal é analisar a variação de milissegundos entre o processamento de sinais de entrada (System Clock) e a execução fonética via `Web Audio API`.

## Technical Specs
- **Engine:** JavaScript ES6+
- **Precision:** Performance.now() integration.
